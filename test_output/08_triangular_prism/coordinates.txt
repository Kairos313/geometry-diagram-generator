# Geometric Blueprint

### Part 1: Geometric Context from Question

#### 1. QUESTION OBJECTIVE
A right triangular prism with base triangle $ABC$ ($\angle ABC = 90^\circ$, $AB = 3$ cm, $BC = 4$ cm) and height $10$ cm. The objective is to find the length of the space diagonal $AF$, where $F$ is the top vertex above $C$.

#### 2. GIVEN ELEMENTS
*   **Given Lengths:** $AB = 3$ cm, $BC = 4$ cm, Prism Height ($AD = BE = CF$) = $10$ cm.
*   **Given Angles:** $\angle ABC = 90^\circ$.
*   **Given Properties:** Right triangular prism (vertical edges are perpendicular to the base).

#### 3. ALL GEOMETRIC ELEMENTS
*   **Points:** $A, B, C$ (base vertices); $D, E, F$ (top vertices).
*   **Lines/Segments:** $AB, BC, CA$ (base edges); $DE, EF, FD$ (top edges); $AD, BE, CF$ (vertical edges); $AF$ (space diagonal).
*   **Angles:** $\angle ABC, \angle DEF, \angle ABE, \angle CBE, \angle DAB, \angle FCB$.
*   **Polygons/Shapes:** Triangle $ABC$ (base), Triangle $DEF$ (top), Rectangles $ABED, BCFE, ACFD$ (lateral faces).
*   **3D Objects:** Right triangular prism.

### Part 2: Coordinate System and Scale Definition (Calculated)

#### 1. ORIGIN PLACEMENT (CRITICAL)
*   Point **B** is chosen as the origin: **(0.000, 0.000, 0.000)**.

#### 2. AXES ALIGNMENT
*   **Primary Axis (X-axis):** The vector from $B$ to $C$ defines the positive X-axis.
*   **Primary Plane (XY-plane):** The base triangle $ABC$ lies on the XY-plane ($Z=0$).
*   **Vertical Axis (Z-axis):** The height of the prism extends along the positive Z-axis.

#### 3. SCALE DEFINITION
*   **Method:** $AB = 3$ cm is set to **5.0 units**.
*   **Scale Reference:** $AB = 3$ cm = $5.0$ units, scale factor = $5/3 \approx 1.667$.
*   $BC = 4$ cm $\times 1.667 = 6.667$ units.
*   Height = $10$ cm $\times 1.667 = 16.667$ units.
