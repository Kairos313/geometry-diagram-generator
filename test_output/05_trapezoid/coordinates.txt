=== GEOMETRIC BLUEPRINT - COORDINATES ===

## Geometric Blueprint

### Part 1: Geometric Context from Question

#### 1. QUESTION OBJECTIVE
Find the height of an isosceles trapezoid $ABCD$ where $AB \parallel CD$, $AB = 16$ cm, $CD = 10$ cm, and $AD = BC = 5$ cm.

#### 2. GIVEN ELEMENTS
*   **Given Lengths:** $AB = 16$ cm, $CD = 10$ cm, $AD = 5$ cm, $BC = 5$ cm.
*   **Given Properties:** $AB \parallel CD$, $ABCD$ is an isosceles trapezoid ($AD = BC$).

#### 3. ALL GEOMETRIC ELEMENTS
*   **Points:** $A, B, C, D$, $D'$ (projection of $D$ onto $AB$ to represent height).
*   **Lines/Segments:** $AB, BC, CD, DA$ (sides), $DD'$ (height).
*   **Angles:** $\angle DAB, \angle ABC, \angle BCD, \angle CDA$ (interior angles), $\angle AD'D$ (right angle for height).
*   **Polygons/Shapes:** Isosceles trapezoid $ABCD$.
*   **Construction Elements:** Perpendicular segment $DD'$ from vertex $D$ to base $AB$.

### Part 2: Coordinate System and Scale Definition (Calculated)

#### 1. ORIGIN PLACEMENT (CRITICAL)
*   Point **A** is the origin: **(0.000, 0.000, 0.000)**.

#### 2. AXES ALIGNMENT
*   **Primary Axis (X-axis):** The vector from $A$ to $B$ defines the positive X-axis.
*   **Primary Plane (XY-plane):** The trapezoid lies on the XY-plane ($Z = 0$).

#### 3. SCALE DEFINITION
*   **Method:** Set the length of the first mentioned side $AB$ (16 cm) to **5.0 units**.
*   **Scale Reference:** $AB = 16$ cm = $5.0$ units. Scale factor = $5/16 = 0.3125$.
*   **Derived Scaled Lengths:**
    *   $CD = 10 \times 0.3125 = 3.125$ units.
    *   $AD = BC = 5 \times 0.3125 = 1.5625$ units.
    *   Horizontal offset for $D$ ($x$): In an isosceles trapezoid, $x = (AB - CD) / 2 = (16 - 10) / 2 = 3$ cm.
    *   Scaled $x = 3 \times 0.3125 = 0.9375$ units.
    *   Height ($h$): $h = \sqrt{AD^2 - x^2} = \sqrt{5^2 - 3^2} = 4$ cm.
    *   Scaled $h = 4 \times 0.3125 = 1.250$ units.
