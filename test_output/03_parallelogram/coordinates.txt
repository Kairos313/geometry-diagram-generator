=== GEOMETRIC BLUEPRINT - COORDINATES ===

## Geometric Blueprint

### Part 1: Geometric Context from Question

#### 1. QUESTION OBJECTIVE
*   Find the length of diagonal $AC$ in a parallelogram $ABCD$ where $AB = 10$ cm, $AD = 6$ cm, and $\angle DAB = 60^\circ$.

#### 2. GIVEN ELEMENTS
*   **Given Lengths:** $AB = 10$ cm, $AD = 6$ cm.
*   **Given Angles:** $\angle DAB = 60^\circ$.
*   **Given Properties:** $ABCD$ is a parallelogram (implies $AB \parallel CD$, $AD \parallel BC$, $AB = CD$, $AD = BC$).

#### 3. ALL GEOMETRIC ELEMENTS
*   **Points:** $A, B, C, D$.
*   **Lines/Segments:** $AB, BC, CD, DA$ (sides), $AC$ (diagonal).
*   **Angles:** $\angle DAB, \angle ABC, \angle BCD, \angle CDA$.
*   **Polygons/Shapes:** Parallelogram $ABCD$.

### Part 2: Coordinate System and Scale Definition (Calculated)

#### 1. ORIGIN PLACEMENT (CRITICAL)
*   Point **A** is the origin: **(0.000, 0.000, 0.000)**.

#### 2. AXES ALIGNMENT
*   **Primary Axis (X-axis):** The vector from $A$ to $B$ defines the positive X-axis.
*   **Primary Plane (XY-plane):** The parallelogram $ABCD$ lies on the XY-plane ($Z = 0.000$).

#### 3. SCALE DEFINITION
*   **Method:** Set the first significant distance $AB = 10$ cm to **5.0 units**.
*   **Scale Reference:** $AB = 10$ cm = $5.0$ units, scale factor = $5/10 = 0.5$.
*   $AD = 6$ cm $\times 0.5 = 3.0$ units.
